## Run Locally

Clone the project

```bash
  git clone git@github.com:Les-Cop1/menu-crous.git
```

Go to the project directory

```bash
  cd menu-crous
```

Install dependencies

```bash
  yarn
```

Start the project

```bash
  yarn start
```

### URLs :

Live version of the project : [http://localhost:3000](http://localhost:3000)
